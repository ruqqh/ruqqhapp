# RUQQH | Software, Systems & Cybersecurity

Professional landing page for RUQQH, a technical collective specializing in custom software development, systems architecture, and cybersecurity solutions.

## Features
- **High-Performance UI**: Built with React and Tailwind CSS.
- **Modern Design**: Brutalist technical aesthetic with smooth Framer Motion animations.
- **Case Studies**: Showcasing real-world projects like Maison Mock.
- **Responsive**: Fully optimized for mobile, tablet, and desktop screens.
- **Automated Deployment**: Includes GitHub Actions for continuous deployment to GitHub Pages.

## Tech Stack
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Hosting**: GitHub Pages

## Getting Started

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run developmental server:
   ```bash
   npm run dev
   ```

## Deployment
This repository is configured to deploy automatically to GitHub Pages when changes are pushed to the `main` branch via the `.github/workflows/deploy.yml` pipeline.
